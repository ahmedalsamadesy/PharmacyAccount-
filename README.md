# نظام الإدارة المالية للصيدلية (Pharmacy Financial Management System)

نظام محاسبي ومالي محلي مستقل (Local-First) يعمل مباشرة عبر المتصفح، مصمم لإدارة الحسابات اليومية، والتدفقات النقدية، وحسابات الموردين، وديون العملاء، ومستحقات الموظفين في الصيدلية بمرونة تامة.

---

## 📌 الميزات والوظائف الرئيسية

* **لوحة التحكم (Dashboard):** تعرض ملخصاً فورياً لرصيد الخزينة، وإجمالي مديونيات الموردين، وجرد ديون العملاء، بالإضافة إلى مخطط بياني لاتجاه المبيعات وآخر العمليات المسجلة.
* **إدارة المبيعات (Sales):** تسجيل شيفتات المبيعات اليومية وتحديد الموظف المستلم والخزينة التي أودعت فيها الأموال.
* **إدارة المشتريات والمرتجع (Purchases & Returns):** إدارة فواتير المشتريات الواردة بالآجل أو الدفع الفوري، مع إمكانية تسجيل مرتجعات السلع وتأثيرها المباشر على حساب المورد.
* **دفتر المصروفات التشغيلية (Expenses):** قيد النفقات التشغيلية والنثريات والرواتب، مع تصنيفها قطاعياً وربطها بالخزائن المختلفة.
* **دفتر حسابات الموردين (Suppliers Ledger):** كشف حساب تفصيلي لكل مورد يوضح الفواتير، والمدفوعات، والارتجاعات، مع رصد الرصيد المستمر، وجدول أسبوعي لتنظيم مواعيد الطلبيات.
* **العملاء والتحصيل (Customers & Receivables):** تسجيل جرد ديون العملاء الإجمالية ومتابعة دفعات التحصيل الجارية.
* **إدارة كادر العمل (Employees):** تسجيل بيانات الموظفين، والتخصصات، والرواتب المعتمدة وساعات العمل.
* **التحويلات النقدية (Cash Transfers):** توثيق حركة نقل النقدية بين الخزائن المختلفة (أو المحافظ الإلكترونية) وتحديد المسؤول عن التحويل.
* **حساب الخزينة الرئيسي (Treasury Book):** دفتر حسابات تفصيلي يسجل حركة الصادر والوارد للرصيد النقدي الفعلي مع احتساب الرصيد الجاري تلقائياً.
* **التقارير والملخصات:** 
  * توليد تقرير الحساب الختامي (قائمة التدفقات النقدية).
  * الملخص المالي والتشغيلي اليومي الشامل والقابل للطباعة.
  * الملخص المالي الشهري لتقييم الأداء المالي.
  * تقارير متقدمة مخصصة مع إمكانية الفلترة والتصدير إلى Excel.

---

## 🛠️ البنية التقنية (Technical Stack)

* **اللغات الأساسية:** HTML5, CSS3 (مع دعم المظهر الداكن والنهاري)، JavaScript (Vanilla JS).
* **قاعدة البيانات:** IndexedDB (عبر المتصفح مباشرة لضمان بقاء البيانات محلياً وبسرعة معالجة عالية).
* **التشفير والحماية:** معيار التشفير الآمن (SHA-256) عبر واجهة المتصفح البرمجية (Native Web Crypto API) لحماية شاشة الدخول المالي.
* **المكتبات الخارجية المضمنة سحابياً (CDNs):**
  * **SheetJS (XLSX):** لتصدير جداول البيانات والتقارير المالية إلى ملفات Excel.
  * **Firebase SDK:** لتفعيل الربط والمزامنة السحابية الاختيارية (Firestore) مع خاصية دمج البيانات بالبصمة الزمنية (Timestamp Merge).

## 🚀 طريقة التشغيل والبدء

1. لا يتطلب البرنامج أي تثبيت أو بيئة عمل معقدة (No Installation required).
2. قم بحفظ الكود البرمجي في ملف بامتداد `.html` (مثال: `index.html`).
3. انقر نقراً مزدوجاً على الملف لفتحه بواسطة أي متصفح ويب حديث (مثل Google Chrome، أو Safari، أو Microsoft Edge).
4. عند فتح البرنامج لأول مرة، يمكنك تعيين إعدادات الصيدلية وتفعيل رمز الحماية PIN الخاص بك من قسم الإعدادات.

---

## ⚠️ تنبيه هام حول سلامة البيانات

* **حفظ البيانات:** نظراً لأن النظام يعتمد على قاعدة بيانات المتصفح المحلية (IndexedDB)، فإن مسح بيانات المتصفح (Clear Cache/Cookies) قد يؤدي إلى حذف البيانات المخزنة.
* **النسخ الاحتياطي:** يوصى بشدة باستخدام زر **"حفظ نسخة احتياطية"** الموجود في الشريط العلوي بشكل دوري (يومياً) للاحتفاظ بملف احتياطي بصيغة JSON على جهازك، أو تفعيل خيار **"المزامنة السحابية مع Firebase"** لضمان حفظ نسخة سحابية آمنة ومزامنتها عند تغيير الأجهزة.
